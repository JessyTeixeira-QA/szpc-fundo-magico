**Magic Background** is an interactive application that uses Artificial Intelligence to transform textual descriptions into animated and stylized backgrounds in real-time. The project combines the power of **Front-end** development with the flexibility of automation via **n8n**.

---

## ğŸ“‹ About the Project

This project was developed during the **Semana do Zero ao Programador Contratado (SZPC)**. The core idea is to allow users to describe a scenario or style (e.g., *"a space gradient with shooting stars"*) and the AI automatically generates the necessary HTML and CSS code to create that visual effect, applying it instantly to the page background.

## âš™ï¸ Workflow Configuration (n8n)

For the project to work, an n8n workflow must be configured as follows:

1. **Webhook Node:** Receives the user's prompt via POST method.
2. **AI Agent / Google Gemini:** Processes the text and generates a JSON object containing `html` and `css` keys.
3. **Code Node:** Cleans the AI response, ensuring only pure code is returned (removing markdown blocks).
4. **Respond to Webhook:** Returns the processed data to the frontend.
